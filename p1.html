Digital Examination System

As project lead, I was responsible for the sucessful design and implementation of a digital examination system for the Nursing and Midwifery Council of Ghana.

Project Period: 2017 - 2019

Project scope: The Nursing and Midwifery Council of Ghana's desire to have the process of licensing all practitioners to be fully automated. This requires that both practical, Care study and written examination componenents are fully digitized.

Project Deliverables:

  • Mobile Application (Android based)
    • A mobile app was designed with rating tools. Examiners assess the students while performing the assigned tasks. Upon submission the marks are transmitted and saved securely to the server
  • Digital Examination Platform (Web based)
    • An open source platform (Moodle) was chosen to run on postgres sql. The webserver used was apache web service.
  • System Documentation
    • Database Dictionary | Project Start Architecture |

Project Achievements: Examination results are published 10 days after examination a reduction from 40 days

product available : https://www.des.nmc.gov.gh
